## Development

### Frontend

**Simple web pages to access APIs**

-   Typescript
-   React
-   Next.js (A React framework)

### Backend

**Handle all the data and access to database**

-   Typescript
-   Node.js
-   Knex (A SQL query builder that is flexible, portable, and fun to use!)

### Database

**Store Data**

-   MySQL

## Installation

### Frontend

DEFAULT PORT 3000

```
cd frontend
npm install
npm run dev
```

localhost:3000

### Backend

DEFAULT PORT 8002

fill ./backend/.env

```
cd backend
npm install
```
